# DigitalNZ Dataset and Notebook Archive

This archive packages the `digitalnz/` notebook collection, supporting facet tables, and related documentation for long-term citation and reuse.

## Source

- Repository: `https://github.com/edithatogo/dnz`
- Upstream collection: `https://github.com/GLAM-Workbench/digitalnz`
- Current released source snapshot: `v1.0.0`
- License: MIT

## Contents

- Jupyter notebooks for DigitalNZ API exploration and harvesting.
- Facet exports under `digitalnz/facets/`.
- Supporting CSV and HTML artifacts used in the notebook workflows.
- Zenodo metadata in `digitalnz/.zenodo.json`.

## Citation

Use the metadata in `digitalnz/.zenodo.json` for Zenodo publication and citation fields.
The notebook README and release metadata should stay synchronized with the archived files.

## Notes

- The archive intentionally keeps the original notebook filenames and facet exports.
- The `digitalnz/` directory is the archival payload for Hugging Face and Zenodo publication.
